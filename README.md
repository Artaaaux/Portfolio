## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/Artaaaaux/Portfolio.git
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

Build production:

```bash
npm run build
```
